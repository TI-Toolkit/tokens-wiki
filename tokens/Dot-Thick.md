# `Dot-Thick`

## Description
Sets dot plotting mode; resets all Y=editor graph-style settings to Dot-Thick.


<b>Availability</b>: Token only available from within the Basic editor.

## Syntax
`Dot-Thick`

## Location
<tt><kbd><b>mode</b></kbd></tt>, `Dot-Thick`

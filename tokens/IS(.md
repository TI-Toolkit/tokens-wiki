# `IS>(`

## Description
Increments `variable` by 1; skips `commandA` if `variable`>`value`.

<b>Comment</b>::commandA,:commands

<b>Availability</b>: Token only available from within the Basic editor.

## Syntax
`:IS>(variable,value) :commandA:commands`

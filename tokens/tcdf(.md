# `tcdf(`

## Description
Computes the Student-`t` distribution probability between `lowerbound` and` upperbound` for the specified degrees of freedom` df`.


<b>Availability</b>: Token available everywhere.

## Syntax
`tcdf(lowerbound,upperbound,df)`

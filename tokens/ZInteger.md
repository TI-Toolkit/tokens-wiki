# `ZInteger`

## Description
Redefines the viewing window using the following dimensions: `TraceStep=1, `Δ`X=0.5, Xscl=10, `Δ`Y=1, Yscl=10.`


<b>Availability</b>: Token only available from within the Basic editor.

## Syntax
`ZInteger`

## Location
<kbd>zoom</kbd>, `ZOOM`, `8:ZInteger`

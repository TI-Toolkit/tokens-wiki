# `getDate`

## Description
Returns a list giving the date according to the current value of the clock. The list is in {year,month,day} format.


<b>Availability</b>: Token available everywhere.

## Syntax
`getDate`

## Location
<kbd>2nd</kbd>, <kbd>catalog</kbd>, `getDate`

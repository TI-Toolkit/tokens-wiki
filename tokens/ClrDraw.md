# `ClrDraw`

## Description
Clears all drawn elements from a graph or drawing.


<b>Availability</b>: Token available everywhere.

## Syntax
`ClrDraw`

## Location
<kbd>2nd</kbd>, <kbd>draw</kbd>, `DRAW`, `1:ClrDraw`

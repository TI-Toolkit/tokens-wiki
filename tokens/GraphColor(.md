# `GraphColor(`

## Description
Sets the color for `function#`.
Color#: 10 - 24 or color name pasted from [vars] COLOR.


<b>Availability</b>: Token only available from within the Basic editor.

## Syntax
`GraphColor(function#,color#)`

<table>

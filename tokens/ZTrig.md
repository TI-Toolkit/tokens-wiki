# `ZTrig`

## Description
Replots the functions immediately, updating the window variables to preset values for plotting trig functions.


<b>Availability</b>: Token only available from within the Basic editor.

## Syntax
`ZTrig`

## Location
<kbd>zoom</kbd>, `ZOOM`, `7:ZTrig`

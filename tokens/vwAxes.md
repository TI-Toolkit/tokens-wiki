# `vwAxes`

## Description
Sets sequence graphs to plot `v(``n``)` on the x-axis and `w(``n``)` on the y-axis.


<b>Availability</b>: Token only available from within the Basic editor.

## Syntax
`vwAxes`

## Location
<kbd>2nd</kbd>, <kbd>format</kbd>, `vw`

# `DiagnosticOff`

## Description
Sets diagnostics-off mode; r, r², and R² are not displayed as regression model results.


<b>Availability</b>: Token available everywhere.

## Syntax
`DiagnosticOff`

## Location
<kbd>2nd</kbd>, <kbd>catalog</kbd>, `DiagnosticOff`

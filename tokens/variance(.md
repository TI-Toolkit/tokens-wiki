# `variance(`

## Description
Returns the variance of the elements in `list` with frequency `freqlist`.


<b>Availability</b>: Token available everywhere.

## Syntax
`variance(list[,freqlist])`

## Arguments
<table>
<tr><th>Name</th><th>Type</th><th>Optional</th></tr>

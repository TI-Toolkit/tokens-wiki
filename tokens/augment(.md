# `augment(`

## Description
Returns a matrix, which is `matrixB` appended to `matrixA` as new columns.


<b>Availability</b>: Token available everywhere.

## Syntax
`augment( matrixA ,matrixB )`

## Arguments

<tr><th>Name</th><th>Type</th><th>Optional</th></tr>

</table>

## Description

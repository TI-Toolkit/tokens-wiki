# `*row+(`

## Overview
Returns a matrix with `rowA` of `matrix` multiplied by `value`, added to `rowB`, and stored in `rowB`.


<b>Availability</b>: Token available everywhere.

## Syntax
`*row+(value,matrix,rowA,rowB)`

# `PlotsOff `

## Description
Deselects all stat plots or one or more specified stat plots (`1`, `2`, or `3`).


<b>Availability</b>: Token available everywhere.

## Syntax
`PlotsOff [1,2,3]`

## Location
<kbd>2nd</kbd>, <kbd>stat plot</kbd>, `STAT PLOTS`, `4:PlotsOff`

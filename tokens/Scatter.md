# `Scatter`

## Description
Used as the "type" argument in the command
Where # gives Plot1, Plot2 or Plot3.


<b>Availability</b>: Token only available from within the Basic editor.

## Syntax
`Plot#(type,Xlist[,freqlist,color#])`

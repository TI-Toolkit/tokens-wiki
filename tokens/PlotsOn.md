# `PlotsOn `

## Description
Selects all stat plots or one or more specified stat plots (`1`, `2`, or `3`).


<b>Availability</b>: Token available everywhere.

## Syntax
`PlotsOn [1,2,3]`

## Location
<tt><kbd><b>2nd</b></kbd></tt>, <kbd>stat plot</kbd>, `STAT PLOTS`, `5:PlotsOn`
